# Income Tax Calculator
### Bangladesh Income Tax Investment Rebate & Payable Tax Calculator

Developed by **Md. Nur Uddin Mahmud** — Income Tax Practitioner  
BBA & MBA, University of Chittagong

---

## Features
- Monthly salary input (July – June fiscal year)
- Tax slab calculation (AY 2026-27 & 2027-28)
- Investment rebate calculator with allowable limit enforcement
- Existing investment tracker with per-category caps (DPS, Sanchayapatra, etc.)
- TDS deduction & monthly TDS estimate
- PDF download with full breakdown

## Deployment
This is a static single-file site. No build step required.

Hosted on Vercel — connected via GitHub.

## Tech
HTML · CSS · Vanilla JavaScript · jsPDF · jsPDF-AutoTable
